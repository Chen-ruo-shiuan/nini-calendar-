純露濕敷教學

Hydrosol Compress Guide — 讓肌膚在保養品底層建立好水合基底

01
潔顏

以溫和潔顏品清洗，確保臉部乾淨，輕輕吸乾水分，不要過度摩擦。

02
先使用保濕原液

選擇 1～2 款保濕型原液（如水溶性 B5、積雪草、神經醯胺），取 5～6 滴輕壓在臉上，待臉部完全吸收。

03
純露浸濕化妝棉

選擇適合膚況的純露（可混合 2 款），充分浸入濕敷化妝棉，輕擰至不滴水。

04
濕敷 3～5 分鐘

平躺放鬆，靜待 3～5 分鐘，讓純露在原液基底上進一步補充水分。

取下化妝棉後，以掌心輕壓臉部讓剩餘純露充分吸收，再接續使用鎖水型原液（如油溶性 B5、神經醯胺）加強鎖水。最後完成乳液或乳霜步驟。建議每週 2～3 次，服務前後加強使用效果更佳。

搭配建議

Pairing Guide — 依膚況選擇最適合的原液 × 純露組合

乾燥 / 敏感肌

原液：神經醯胺 + 積雪草 + 水溶性B5 + 油溶性B5（鎖水）
純露：千葉玫瑰 / 洋甘菊 / 橙花 / 桂花
著重屏障修護、舒緩補水與持久鎖水

油性 / 混合肌

原液：兒茶素 + B3（油溶）
純露：茶樹 / 葡萄柚 / 橙花
著重調理皮脂、收斂毛孔狀態

暗沉 / 均勻膚色

原液：穀胱甘肽 + 維他命C（油溶）+ B3（油溶）
純露：橙花 / 千葉玫瑰 / 桂花
著重抗氧化與膚色均勻調護

熟齡 / 彈力調護

原液：藍銅胜肽 + EGF + 膠原蛋白 + 油溶性B5
純露：永久花（季節）/ 千葉玫瑰 / 沒藥（季節）
著重肌膚彈力、緊緻感調護

服務後安撫調護

原液：積雪草 + 神經醯胺 + 水溶性B5
純露：洋甘菊 / 金盞花（季節）/ 橙花
著重服務後肌膚的舒緩與穩定

情緒放鬆 / 睡前調護

原液：油溶性B5 + 膠原蛋白 + EGF
純露：岩蘭草（季節）/ 千葉玫瑰 / 桂花
著重晚間深層滋潤與身心放鬆

早晚使用順序

Daily Routine — 原液與純露的正確疊加順序

早晨保養

1
溫水潔顏（早晨輕柔清潔）
2
純露輕壓（化妝水步驟，可選擇性濕敷）
3
抗氧化原液（穀胱甘肽 / 維他命C油溶 / B3油溶）
4
保濕原液（水溶性B5 / 積雪草）
5
鎖水原液（神經醯胺 / 油溶性B5）
6
乳液或乳霜（加強鎖水）
7
防曬（每日必要步驟）

晚間保養

1
卸妝或潔顏（視使用的卸妝產品，若含乳化成分可單次；若僅潔顏，建議先以卸妝品去除全臉彩妝再潔顏）
2
純露輕壓或進行濕敷（視膚況選擇）
3
調護性原液（積雪草 / EGF / 藍銅胜肽）
4
保濕修護原液（水溶性B5 + 神經醯胺）
5
鎖水原液（油溶性B5 / 膠原蛋白）
6
乳液或乳霜（加強鎖水）
7
眼霜（眼周區域）

原液疊加原則：質地由稀至濃、水溶在前、油溶在後。 每款原液使用後待臉部完全吸收再疊加下一款。每次使用 2～3 款原液為宜，避免過度疊加造成肌膚負擔。水溶性 B5 建議用於補水步驟，油溶性 B5 建議用於最終鎖水步驟。
